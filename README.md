# 🚀 CEO 매출 향상 전략 대시보드 (Legendary Edition)
> **"데이터를 지배하는 자가 비즈니스를 지배한다."**
> 50개의 전문 분석 탭을 통해 경영의 모든 해답을 제시하는 비즈니스 인텔리전스 시스템입니다.

---

## 📌 1. 프로젝트 개요
이 프로젝트는 네이버 스마트스토어 등 이커머스 판매 데이터를 기반으로, **MD, 마케팅, 물류, 경영 전략** 등 비즈니스의 전 영역을 심층 분석하는 Python streamlit 기반의 대시보드입니다. 단순한 리포팅을 넘어, AI 기반 예측과 전략적 제언을 통해 사장님의 의사결정을 돕는 '경영 파트너' 역할을 수행합니다.

## 🛠 2. 주요 기능 (Legendary 50 Tabs)
본 대시보드는 총 50개의 전문 분석 탭으로 구성되어 있습니다.

### [A] 경영 및 매출 (Management & Sales)
*   **통합 관제 센터**: 비즈니스 핵심 지표 실시간 통제 (매출, 이익, 활성 유저 등)
*   **AI 경영 비서**: 생성형 AI가 요약해 주는 일일 경영 브리핑
*   **스마트 목표 트래커**: 월 매출 목표 대비 달성률 및 예측
*   **TOP 10 분석**: 상품, 셀러, 지역별 매출/이익 상위 랭킹

### [B] 상품 및 전략 (Merchandising & Strategy)
*   **상품 ABC 분석**: 파레토 법칙에 기반한 상품 등급 분류
*   **성장 매트릭스 (BCG)**: Star, Cash Cow, Dog 상품 포트폴리오 진단
*   **가격 저항성 분석**: 가격 인상에 따른 판매량 변화 측정
*   **연관 구매 네트워크**: 함께 팔리는 상품 간의 시너지 시각화

### [C] 고객 및 마케팅 (Customer & Marketing)
*   **VIP 프로파일링**: 최상위 고객 1:1 심층 분석 (Persona CRM)
*   **AI LTV 예측**: 미래의 VIP를 발굴하는 머신러닝 모델링
*   **고객 생존 분석**: 시간 경과에 따른 이탈 생존율(Retention) 추적
*   **멀티채널 기여도**: 채널별 고객 획득(First) 및 전환(Last) 기여도 비교

### [D] 운영 및 리스크 (Operation & Risk)
*   **골든/데드 크로스**: 매출 추세 반전 타이밍 포착 (매수/매도 시그널)
*   **재고 효율 분석**: 잘 안 팔리는 악성 재고 식별
*   **취소/반품 리스크**: 고객 불만 및 손실 비용 분석
*   **물류 효율 분석**: 배송 리드타임 병목 구간 진단

## 💻 3. 실행 방법 (How to Run)

### 1) 필수 환경 설정
Python 3.8 이상이 필요하며, 아래 라이브러리를 설치해야 합니다.
```bash
pip install streamlit pandas numpy plotly statsmodels
```

### 2) 데이터 준비
`data/` 폴더 안에 판매 데이터 CSV 파일을 위치시키세요.
(자동으로 폴더 내의 모든 CSV를 읽어 통합 및 중복 제거를 수행합니다.)

### 3) 대시보드 실행
터미널에서 아래 명령어를 입력하세요.
```bash
streamlit run app_sales.py
```
